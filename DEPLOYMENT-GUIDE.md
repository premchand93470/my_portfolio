### Step 1: Create GitHub Account (if you don't have one)

### Step 2: Create a New Repository

### Step 3: Upload Your Files
**Option A: Using GitHub Desktop (Easiest)**
1. Download GitHub Desktop: https://desktop.github.com
2. Install and sign in
3. File â†’ Add Local Repository â†’ Select your `portfolio-main` folder
4. Click "Publish repository"
5. Make sure to check "Keep this code private" is **UNCHECKED** (must be public)

**Option B: Using GitHub Website (Drag & Drop)**
1. Go to your new repository on GitHub
2. Click "uploading an existing file"
3. Drag and drop ALL these files/folders:
   - `index.html`
   - `images/` folder (with indiajoy-stage.jpg.png inside)
   - `script.js`
   - `styles.css`
4. Click "Commit changes"

### Step 4: Enable GitHub Pages
1. Go to your repository on GitHub
2. Click "Settings" tab
3. Scroll down to "Pages" in left sidebar
4. Under "Source", select "Deploy from a branch"
5. Branch: `main` (or `master`)
6. Folder: `/ (root)`
7. Click "Save"
8. Wait 1-2 minutes

### Step 5: Access Your Portfolio
Your portfolio will be live at:
`https://[your-username].github.io/portfolio/`

**Example:** If your username is `premchand`, your URL will be:
`https://premchand.github.io/portfolio/`
